Примеры запросов

Все задачи проекта:

SELECT id FROM issues WHERE project_id = 1

Только Task и Bug команды:

SELECT id FROM issues
WHERE type IN ('task', 'bug') AND team_id = 2

Мои задачи (быстрый фильтр):

SELECT id FROM issues
WHERE assignee_id = :current_user_id

Незакрытые баги:

SELECT i.id FROM issues i
JOIN statuses s ON s.id = i.status_id
WHERE i.type = 'bug' AND NOT s.is_done
